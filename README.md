# Tarea2-Sistemas-Operativos

### Instrucciones de compilación

## Parte 1

## Ingrese a la carpeta de la parte 1
```bash
cd ./act1
```

## Compilación
Ejecute el siguiente comando en la terminal:

```bash
make barrera
```

## Ejecución 
### Para ejecutar el programa ingrese:
```
./barrera "cantidad de hebras" "numeros de etapas"
```
